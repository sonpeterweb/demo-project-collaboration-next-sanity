# Portfolio screenshots

Capture these after deploying and seeding with `npm run seed:sanity:fresh`:

| File | What to capture |
|------|-----------------|
| `home.png` | Home page with hero promo and feature highlights |
| `blog.png` | Blog listing with seeded posts |
| `admin.png` | Admin panel (blog or testimonials CRUD) |
| `preview.png` | Preview mode indicator on a draft page |

Then uncomment the image block in the root [README.md](../../README.md).
